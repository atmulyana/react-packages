# @react-packages/ellipsis
It's React component that gives an ellipsis mark on an overflow text without needing to set a fixed
length. It can comportably be used in a responsive design.

Please see this [documentation](https://atmulyana.github.io/react-packages/ellipsis/) and this
[`example`](https://github.com/atmulyana/react-packages/tree/main/example/app/routes/ellipsis.tsx).
