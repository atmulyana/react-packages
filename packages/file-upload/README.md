# @react-packages/file-upload
React component which extends HTML file input by adding a view for the uploaded file(s).
The view is to make sure that we're uploading the correct file(s).

![File Upload Video](https://atmulyana.github.io/react-packages/file-upload/input.gif)

Please see this [documentation](https://atmulyana.github.io/react-packages/file-upload/) and this
[example](https://github.com/atmulyana/react-packages/tree/main/example/app/routes/file-upload.tsx).
